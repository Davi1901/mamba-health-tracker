 Mamba Health Tracker
SPA para Controle de Saúde e Performance

Projeto desenvolvido para a disciplina de Desenvolvimento Web, focado em monitoramento de nutrição e ciclos de jejum intermitente.

 Tecnologias Utilizadas
Vue.js 3 (Composition API)

Pinia (Gerenciamento de estado global)

Tailwind CSS (Design responsivo e UX/UI)

Chart.js (Visualização de dados dinâmica)

 Funcionalidades
Módulo de Nutrição: Registro de refeições com barra de progresso reativa.

Módulo de Jejum: Timer para controle de janelas (16:8, 18:6, 20:4).

Dashboard Semanal: Gráfico de consumo calórico e média dos últimos 7 dias.

Persistência Local: Dados salvos via localStorage (não se perdem ao atualizar)
